## Installation
Add this line to your application's Gemfile:

```ruby
gem "med_pipe"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install med_pipe
```

### migrationファイルの追加

```shell
$ rails med_pipe:install:migrations
```
